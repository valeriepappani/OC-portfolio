Kasa

Création d'une application web de location immobilière

  • Sass
  • React
  • Node.js
  • Front-end

Présentation du client :

Kasa est une entreprise de location d’appartements entre particuliers, avec plus de 500 annonces postées chaque jour, ce qui en fait l'un des leaders dans son secteur en France. Leur volonté était de réaliser une refonte totale.

Projet :

Le projet consistait en une refonte complète du site de Kasa, avec une architecture JavaScript moderne, React pour le front-end et Node.js pour le back-end. Le front-end devait être développé avant le back-end, en utilisant des données temporaires fournies dans un fichier JSON. La mission incluait la création d'une interface responsive basée sur des maquettes Figma, avec une attention particulière portée aux composants et aux guidelines de développement fournies par l’entreprise.

Fonctionnalités principales

  • Développement de l’interface en React, avec des composants et des routes gérées par React Router
  • Intégration de maquettes Figma, y compris les ressources comme les logos et icônes
  • Gestion des "Collapse" pour afficher ou cacher des sections du contenu
  • Animation des menus déroulants et des autres éléments interactifs (selon la maquette Figma)
